# EA Skills

Enterprise Architecture Skills für [GitHub Copilot](https://github.com/features/copilot) — spezialisierte Fähigkeiten für Architektur-Dokumentation, IT-Vertragsanalyse und IT-Lösungsbewertung im EU/DACH-Kontext.

## Installation

```bash
npx skills add gtonic/ea-skills
```

## Enthaltene Skills

| Skill | Beschreibung |
|---|---|
| **c4-architecture** | Generiert C4-Architekturdiagramme (System Context, Container, Component, Deployment, Dynamic) in ASCII-Notation |
| **it-contract-analysis** | Quantitative IT-Vertragsrisikoanalyse mit Ampelsystem + Risikoscore für DACH/EU-Recht (DSGVO, NIS2, AI Act) |
| **it-solution-assessment** | Strukturierte Bewertung von IT-Lösungen (SaaS, On-Prem, Hybrid) mit gewichtetem Scoring und K.O.-Kriterien |

Detaillierte Dokumentation: [.skills/Readme.md](.skills/Readme.md)
